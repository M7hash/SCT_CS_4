# Simple Python Keylogger with Date &amp; Time  
A Python-based keylogger designed for educational and ethical cybersecurity purposes. This project demonstrates how keystrokes can be captured and logged with timestamps, helping learners understand keylogging concepts, keyboard event handling, and file logging in Python.
  
  # Features :
- Logs all keyboard input
- Records **date and time** for each keypress
- Saves logs to `logs.txt`
- Beginner-friendly Python code
